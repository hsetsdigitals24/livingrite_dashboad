## Responsive Behavior

### Mobile (< 768px)
- Single column layouts
- Full-width cards
- Stacked sidebar (bottom)
- Collapsed navigation
- Touch-friendly spacing

### Tablet (768px - 1024px)
- 2-column grids
- Sidebar on right (narrower)
- Medium-sized images
- Adjusted fonts

### Desktop (> 1024px)
- 3-column grids
- Sticky sidebar
- Large hero images
- Full-featured layout
